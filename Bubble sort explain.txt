# **7ÔłŹ‚É£ Decide if swap is needed**

boolean swap = false;
switch (type) {
    case 1: // ID
        swap = compareString(p1.getId(), p2.getId(), ascending);
        break;
    case 2: // Name
        swap = compareString(p1.getRecipientName().toLowerCase(),                       p2.getRecipientName().toLowerCase(),ascending);
        break;
    case 3: // Weight
        swap = compareDouble(p1.getWeight(), p2.getWeight(), ascending);
        break;
    default:
        System.out.println("Invalid sort type!");
        return;
}
```

### ūüĒĻ How swapping decision works:

1. compareString ‚Üí returns true if elements need to be swapped based on ascending/descending
2. .toLowerCase() ‚Üí ensures case-insensitive sort for names
3. compareDouble ‚Üí for numerical comparison of weights

> Pang-defense: separates comparison logic from actual swap, makes code clean.
```

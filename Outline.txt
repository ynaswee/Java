# 🔵 IMPLEMENTATION NG SORTING ALGORITHM

## 🟣 SLIDE 6: What is Bubble Sort?

* Definition:

  * Compares adjacent elements
  * Swaps if needed
* Repeats until sorted

---

## 🟣 SLIDE 7: Bubble Sort Logic (Simple Flow)

* Loop through list
* Compare adjacent items
* Swap if condition is true
* Repeat

👉 Pwede ka maglagay ng simple diagram dito

---

## 🟣 SLIDE 8: Bubble Sort in Code

* Show snippet:

```java
for (int i = 0; i < packages.size() - 1; i++) {
    for (int j = 0; j < packages.size() - i - 1; j++) {
        if (swap) {
            // swap logic
        }
    }
}
```

* Explain:

  * outer loop = passes
  * inner loop = comparisons

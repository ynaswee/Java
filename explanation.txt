# 🔵 SORTING ALGORITHM

## 🟣 SLIDE 6: WHAT IS BUBBLE SORT

Bubble Sort is a sorting algorithm that:

* Compares adjacent elements
* Swaps them if needed
* Repeats the process until sorted

---

## 🟣 SLIDE 7: HOW BUBBLE SORT WORKS

Steps:

1. Compare two adjacent elements
2. Swap if they are in the wrong order
3. Move to the next pair
4. Repeat until no swaps are needed

---

## 🟣 SLIDE 8: BUBBLE SORT CODE

```java
for (int i = 0; i < packages.size() - 1; i++) {
    for (int j = 0; j < packages.size() - i - 1; j++) {
        if (condition) {
            // swap elements
        }
    }
}
```

* Outer loop = number of passes
* Inner loop = comparisons

## 🔹 STEP 2: Double Loop (Bubble Sort)

```java
for (int i = 0; i < packages.size() - 1; i++) {
    for (int j = 0; j < packages.size() - i - 1; j++) {
```

### 💡 Meaning:

#### Outer loop (`i`)

👉 Ilang beses uulit ang sorting
👉 (n-1 passes)

#### Inner loop (`j`)

👉 Nagko-compare ng **magkatabing elements**

---

## 🔹 STEP 3: Get objects

```java
Package p1 = packages.get(j);
Package p2 = packages.get(j + 1);
```

👉 Magkatabing packages

---

## 🔹 STEP 4: Decide kung mag-swap

```java
boolean swap = false;

switch(type) {
    case 1: swap = compareString(p1.getId(), p2.getId(), ascending);
    case 2: swap = compareString(name, name, ascending);
    case 3: swap = compareDouble(p1.getWeight(), p2.getWeight(), ascending);
}
```

---

## 🔹 STEP 5: Compare Logic

```java
return asc ? a > b : a < b;
```

### 💡 Meaning:

| Order      | Condition       |
| ---------- | --------------- |
| Ascending  | swap if p1 > p2 |
| Descending | swap if p1 < p2 |

---

## 🔹 STEP 6: Swap

```java
if (swap) {
    packages.set(j, p2);
    packages.set(j + 1, p1);
}
```

👉 Pinagpapalit ang position

# 🧠 Paano gumagana ang bubble sort (actual concept)

## 🔥 IDEA:

> Pinapa-"bubble" pataas ang pinakamalaking value bawat pass

---

## 🔹 Example (weight sorting ascending)

Initial:

```
[15, 12, 8]
```

---

### 🔁 PASS 1:

Compare:

```
15 vs 12 → swap → [12, 15, 8]
15 vs 8  → swap → [12, 8, 15]
```

👉 Largest (15) nasa dulo na

---

### 🔁 PASS 2:

```
12 vs 8 → swap → [8, 12, 15]
```

---

### ✅ Final:

```
[8, 12, 15]
```

| `get(j)`    | current          |
